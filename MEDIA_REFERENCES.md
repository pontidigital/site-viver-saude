## Instrucoes para Download

1. Acesse cada URL com uma conta Freepik Premium
2. Baixe na maior resolucao disponivel (preferencialmente JPG ou PNG)
3. Otimize as imagens para web (WebP/AVIF quando possivel) usando ferramentas como Squoosh ou Sharp
4. Renomeie conforme a estrutura de pastas em `public/images/` e `public/media/`
5. Videos devem ser comprimidos para web (H.264, bitrate ~2-4Mbps, 1080p)
6. Atualize os caminhos no arquivo `src/lib/constants/media.ts` apos posicionar os arquivos
